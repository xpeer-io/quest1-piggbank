# piggbank

Dashboard financeiro para PMEs. Exibe métricas de faturamento, despesas e fluxo de caixa com filtros por período.

## Requisitos

- Node.js 20+
- npm 10+

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).

## Testes

```bash
# Rodar testes
npm test

# Rodar com coverage
npm test -- --coverage
```

## Build

```bash
npm run build
```
