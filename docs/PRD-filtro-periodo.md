# PRD -Filtro de Período

## Objetivo 
Permitir que o usuário selecione um intervalo de datas para filtrar as métricas do dashbord financeiro.

## Funcionalidade 
- Seleção de data inicial e final
- Atualização automática das métricas
- Aplicaçãodo filtro em todo o dashbord

## Métricas impactadas
- Faturamento
- Despesas
- Lucro líquido
- Transações

## Requisitos
- Interface simples e intuitiva
- Resposta rápida
- Compatibilidade com dados existentes

## Riscos
- Problemas de performanceao filtrar grandes volumes de dados
- Inconsistência de timezone
- Interpretação incorreta dos dados pelo usuário

## Melhorias futuras
- Comparação entre períodos
- Filtro rápido (últimos 7d= dias, 30 dias, etc)
